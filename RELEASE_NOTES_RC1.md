# Kommunicate Newsletter Studio - Release Candidate 1

## Included

- Excel-first newsletter generation remains the default workflow.
- Supporting PDF, Word, Excel and text ingestion.
- Manual snippet-to-placeholder mapping with replace, append and undo.
- AI-assisted mapping from explicitly selected supporting sources.
- Source evidence, confidence, review queue and protected application of suggestions.
- Design controls, branded HTML and Word export, review links and review email.
- Portable project export/import for team handoff.
- Field-level source and provenance CSV report.
- In-app release-readiness diagnostics.
- Updated deployment, acceptance and rollback documentation.

## Not included

- User login, role-based access or a shared cloud workspace.
- Central server-side draft storage.
- OCR for scanned image-only PDFs.
- Formal legal-review approval workflow.

These items require an authenticated backend and should be treated as a later operationalisation phase rather than blockers for controlled internal deployment.
